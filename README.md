# Lab Activity 2: Strings, Lists, Tuples, and Dictionaries

## Description
This program is a menu-driven Python application that manages student records.  
It allows the user to add, view, and update student information.

Each student contains:
- Name
- Course
- Age

The program demonstrates the use of:
- Strings (name, course)
- List (collection of students)
- Tuple (name and course)
- Dictionary (student record)

## Features
1. Add Student (Create)
2. View Students (Read/Display)
3. Update Student (Update)
4. Exit Program

## Default Data
The program starts with three students:

1. De la Cruz | BS CpE | 19  
2. Sarah | BS ECE | 20  
3. Mira | BS ME | 21  

## How to Run
1. Open terminal in the project folder
2. Run the program using:

   python src/main.py
